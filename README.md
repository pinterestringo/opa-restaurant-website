# OPA Greek Street — Restaurant Website

🔗 **[View Live Site](https://pinterestringo.github.io/opa-restaurant-website)**

## Overview
A fully responsive restaurant website for OPA Greek Street, a Greek street food 
restaurant in Shkodër, Albania. Built with vanilla HTML, CSS, and JavaScript — 
no frameworks, no libraries.

## Features
-  **Bilingual** — full English / Albanian (Shqip) language toggle
-  **Interactive menu** with category filtering (wraps, bowls, sides, drinks, etc.)
-  **Responsive design** — works on mobile and desktop
-  **Animations** — smooth scrolling, marquee banner, spinning badge
-  **Custom design system** — consistent color palette and typography 
  using CSS variables and Google Fonts (Playfair Display, Syne, DM Sans)
